# 구매/판매, 컨텐츠 관리 기능

<br>

## 아이템 구매

- 체크 사항
  - 구매 수량 1 이상
  - 포인트 보유 (구매 비용 < 보유 포인트)


- Food / Consumable 차이
    - Food : ItemInventory에 4개까지만 보유할 수 있도록 제어
    - Consumable : 보유 갯수 제한 없음

<br>

## 캐릭터 판매

- 체크 사항
  - 판매 수량 1 이상
  - 캐릭터 보유 여부 (판매 수량 < 보유 수량)

<br>

## 컨텐츠 관리

- 관리
  - 대상 : 아이템, 캐릭터, 팔레트
  - 권한 : 생성, 조회, 수정, 삭제


- 권한 보유자
  - Admin 권한 (Jwt 파싱하여 Admin 권한 가지고 있는지 점검)
